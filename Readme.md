powder manages [Pow](http://pow.cx/)

# Usage #

    $ pow
    => link the current dir_name to ~/.pow/dir_name

    $ pow link bacon
    => link the current dir to ~/.pow/bacon

    $ pow list
    => list all the current apps linked in ~/.pow
    # aliased as pow -l

    $ pow restart 
    => restart the current app
    # aliased as por -r

    $ pow remove
    => unlink the current dir

    $ pow remove bacon
    => unlink bacon
